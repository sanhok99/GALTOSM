In case the Reviewer do not have CADP installed. These log files holds the running time data for the CADP for the model checking in CADP.

In each CASE STUDY we have noted 5 time requirements

1. .lnt -> .bcg
2. .bcg -> .aut
3. .aut--manual_editing--> .aut
4. .aut -> .bcg
5. model check on .bcg

On different size of the models, we have noted the times.

For the reference of the sizes, look into the tables in the GALTOSM_handbook pdf.
