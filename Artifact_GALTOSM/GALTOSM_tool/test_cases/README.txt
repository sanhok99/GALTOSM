FOLDER DETAIL
____________________________

FOLDER 1: adtmc_to_sdtmc
	Contains example .aut files to feed into the GALTOSM's ADTMC to SDTMC tool.
	
FOLDER 2: sdtmc_to_adtmc
	Contains example .sta .tra .lab files which are the inputs to the GALTOSM's SDTMC to ADTMC tool.
	
FOLDER 3: example_logics
	Contains example logics of varius types for the logic translator tool of GALTOSM.
