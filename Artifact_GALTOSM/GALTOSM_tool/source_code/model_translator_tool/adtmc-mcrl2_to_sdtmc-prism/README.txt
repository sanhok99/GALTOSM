FOLEDER DETAILS
_______________________

This folder contains the source code of the model translator tool which translates a ADTMC to an ADTMC.

It accepts an ADTMC generated by the mcrl2 tool in the form of .aut file.
It produces an SDTMC in the form of source code for STORM and PRISM, with the extension .pm.

Note: for running this tool, go to test_case section of thee GALTOSM tool.
