FOLDER DETAIL

This folder contains the Source code for both the Model Translators

1)FOLDER 1: adtmc_to_sdtmc-dev
	Contains
		1) C++ code for ADTMC to SDTMC
		2) The executable of the tool.
		
2)FOLDER 2: sdtmc_to_adtmc
	Contains source code to translate a SDTMC to a ADTMC
	
	
3) FOLDER 3: mcrl2_to_prism
	Contains the python code which translates a .aut(mcrl2 format)  and .pm code.
